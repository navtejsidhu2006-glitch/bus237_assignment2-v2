What this site does

This site is a simple inventory tracker. Users can add items with quantities, see them displayed in a table, and view the same data in a chart. The inventory persists using browser local storage, so it stays after refresh. A “Clear All” button removes all items and wipes local storage.

Extra features

  • Dark mode toggle to suit visual needs.
  • Low stock highlighting (quantity < 10) to make it easy to spot items that need restocking.
